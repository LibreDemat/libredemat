Pièces justificatives

Pour obtenir les informations, il convient de fournir le justificatif demandé. Documents joints : le demandeur joint un nouveau document. Documents disponibles : le demandeur a déjà en réserve, dans son espace foyer, le document demandé. Il le sélectionne pour le joindre à la demande. Vous pouvez numériser vos documents avec :

  • - Un scanner,

  • - un appareil photo numérique,

  • - votre téléphone portable, si il est équipé d'un appareil photo.

Si vous ne disposez pas de moyen de numériser vos documents, vous pouvez apporter vos documents originaux en mairie.
