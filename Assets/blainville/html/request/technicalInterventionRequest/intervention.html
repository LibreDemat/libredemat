Quel service municipal est concerné par votre demande ?

Votre demande est instruite par les services techniques.

Les missions des services techniques municipaux

Les services techniques ont pour mission d'entretenir le patrimoine communal (bâtiments, équipements divers), la voirie, les espaces verts. Ils ont en charge le nettoyage, l'embellissement, le contrôle des installations techniques sur le domaine public et la surveillance des sites.

La nature de votre demande d'intervention

Fournissez une description suffisante permettant aux services techniques de prévoir, dans leur planning d'intervention, le temps et le matériel nécessaire.
